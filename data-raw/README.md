# README

Details on these scripts and what they do.

```
.
├── combine_and_save.R        <-  Final script to run to combine data
├── pull_archived_data.R      <-  WIP script to pull data from Archive.org
├── pull_js_trees_data.R      <-  WIP script to pull data JS in the site
├── pull_live_data.R          <-  Main script to pull data from the garden page
├── README.md
├── screenshot_tree_status.R  <-  Uses headless browser to screenshot the garden
└── treepositions.R           <-  Pulls coordinates of the trees on the page
```

All of these scripts are designed to run on the command line, in addition to
through a GUI or IDE like RStudio or Positron. To run:

```
Rscript insert_script_here.R
```
