# Articles

### All vignettes

- [Plot cherry blossom bloom
  statuses](articles/plot_cherry_blossom_bloom.md):

  This vignette demonstrates how to use the `bbggplots` package to
  visualize the bloom statuses of cherry trees in the Brooklyn Botanic
  Garden. We will use custom SVG icons to represent different bloom
  stages and plot them on a map of the garden.
